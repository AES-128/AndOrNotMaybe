# AndOrNotMaybe

## Usage

Run the program then input a boolean expression

For example:

```bash
$ python and_or_not_maybe
[and_or_not_maybe] >>> a & b

| a | b | Q |
| 0 | 0 | 0 |
| 1 | 0 | 0 |
| 0 | 1 | 0 |
| 1 | 1 | 1 |
```

## Input format

- AND: `&`
- OR: `|`
- NOT: `!` (unary operator)
